mojjss Focus Studio v5.2 drop-in patch

1. Close the desktop app.
2. Extract this ZIP into your existing project root and replace files.
3. Keep your existing desktop_app/config.json, focus_history.db, and schedule.csv.
4. Start the desktop app.
5. Set:
   Camera URL: https://camera.mojjss.ir
   Allowed origins: https://timer.mojjss.ir, https://camera.mojjss.ir
   Require Tailscale identity headers: Off
6. Commit and push the replaced cloudflare_dashboard/public files so Cloudflare Pages redeploys.
