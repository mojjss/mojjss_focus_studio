Mojjss Focus Studio v5.6 - Start Here

1. Close Focus Studio.
2. Run INSTALL_UPDATE_KEEP_DATA.bat.
3. Keep the default target unless your installation is elsewhere:
   D:\my_projects\mojjss_focus_studio
4. Start the app with START_FOCUS_STUDIO.bat.
5. Open the new Diagnostics page in the desktop app.
6. For automatic tunnel recovery, run INSTALL_TUNNEL_WATCHDOG.bat as Administrator.
7. If the tunnel is currently Down/1033, run REPAIR_CLOUDFLARE_TUNNEL.bat.
8. If a problem remains, run FOCUS_STUDIO_DIAGNOSIS.bat and upload the ZIP
   created on your Desktop.

SECRET WARNING
The repair tool may ask for the long Cloudflare tunnel token beginning with eyJ.
Paste it only into the private hidden prompt on your own PC. Never send it in
chat, add it to GitHub, place it in README, or include it in screenshots.

DATA SAFETY
The updater does not overwrite config.json, SQLite databases, schedule.csv,
timer_state.json, exports, readable data, the existing virtual environment, or
Git history. The diagnosis and watchdog do not reset v2rayN, Tailscale,
Winsock, adapters, DNS, firewall rules, or Windows proxy settings.
