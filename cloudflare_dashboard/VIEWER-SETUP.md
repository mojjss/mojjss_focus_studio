# Normal viewer setup

No Tailscale app or Google sign-in is required.

1. Open `https://timer.mojjss.ir` in a normal browser.
2. Enter the viewer key supplied by the owner (`DASHBOARD_VIEWER_KEY`).
3. To view the live camera, enter the separate camera password and press
   **Connect camera**.

Viewer mode receives a smaller, privacy-reduced recent-session list. Session
notes and source fields are omitted.

The dashboard itself remains available when the owner's PC is offline, but its
last snapshot becomes stale. The live camera requires the owner's desktop app
and Cloudflare Tunnel service to be online.
